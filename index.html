        # Projet de jeu de devinette

Date d'exécution : 2024-11-19  
Nombre de lignes de code dans guessinggame.sh : 24  

## Description

Ce projet consiste à créer un jeu où l'utilisateur doit deviner le nombre de fichiers présents dans le répertoire courant. Le programme donne des indications si la réponse est trop basse ou trop haute.

## Instructions

- Exécutez le script bash avec la commande `bash guessinggame.sh`.
- Devinez le nombre de fichiers dans le répertoire courant.
- Le programme continue de demander jusqu'à ce que vous deviniez le bon nombre de fichiers.
